# Benchmark Ladder

This repository now has several synthetic benchmark tiers.

## Tiers

### Tiny

- fast sanity checks
- good for local smoke runs

### Medium synthetic

- broader than tiny
- useful for quick comparison loops

### Clustered

- better for checking neighborhood preservation
- useful for comparing approximation families on structured data

### XLarge synthetic

- widest synthetic tier currently in the repo
- useful for more stressful comparison runs while remaining lightweight

## Suggested order

1. tiny showcase path
2. quantizer comparison
3. clustered benchmark
4. xlarge benchmark
